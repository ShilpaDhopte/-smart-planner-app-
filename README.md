# Smart Planner App

A modern Ubuntu Touch productivity application built using **QML**, **Lomiri Components**, and **QtQuick**.

This application combines multiple productivity tools into one clean mobile-friendly interface.

---

# Features

## Daily Planner

* Add daily tasks
* Task description support
* Time and date management
* Mark tasks as completed
* Delete tasks
* Dynamic task count
* Scrollable task list

---

## Habit Tracker

* Add habits
* Track daily habit completion
* Habit streak counter
* Daily goal tracking
* Progress bar
* Delete habits
* Completion statistics

---

## Expense Tracker

* Add expenses
* Expense categories
* Notes for expenses
* Automatic total expense calculation
* Expense count tracking
* Delete expenses
* Real-time UI updates

---

# Technologies Used

* QML
* QtQuick 2.4
* Lomiri Components
* QtQuick LocalStorage
* Clickable
* Ubuntu Touch

# Installation

## Clone Repository

```bash id="rd2"
git clone https://github.com/apeksha-code-maker/smart-planner-app.git
```

---

## Open Project

```bash id="rd3"
cd smart-planner-app
```

---

## Build and Run

```bash id="rd4"
clickable desktop
```

---

# Requirements

## Install Clickable

```bash id="rd5"
pip3 install clickable-ut
```

---

## Install Ubuntu Touch Dependencies

```bash id="rd6"
sudo apt install qmlscene qtdeclarative5-dev-tools
```
